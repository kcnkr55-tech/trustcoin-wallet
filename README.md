# TrustCoinWallet – Full‑Stack Prototype

A simulated cryptocurrency wallet dashboard with admin controls, KYC, support tickets, and withdrawal simulation.

## Features

- User registration & login
- Shared wallet addresses (BTC, ETH, USDT) for all users
- Dashboard with balances and transaction history
- Withdrawal simulation with gas fee and KYC requirement
- Support ticket system with admin replies
- Admin panel: user management, KYC approval, unlimited fund transfers
- Dark / light theme toggle
- Notification bell

## Deploy

Backend: Render  
Frontend: Vercel
